GitKraken

 - von Axosoft
 - braucht _kein_ installiertes Git
 - download, install, run
 - läuft nativ auf Windows 7+, Mac, Linux
 - sehr aktiv, neue Releases etwa alle 2-4 Wochen
 - NICHT open source!
 
 
Axosoft

- stellt eigentlich Team Collaboration Software her. Die Software versucht ein besseres Jira zu sein, mit Ticketing-System, grafischen Auswertungen, Releaseplanner, et cetera.
- wird von einer Reihe bekannter Unternehmen genutzt, z.B. Nike, Siemens, Verizon, Disney, Intel, ...
- recht kleines Unternehmen (ca. 50 Mitarbeiter)
- sehr female-friendly: frauengeführt, setzen sich auffällig für mehr Frauen in der IT ein
 
Features

 - doppelklick zum auschecken
 - supersimples branch-management
 - drag & drop
 - one-click undo, redo, discard local changes
 - eingebautes merge-tool
 - Volltextsuche auf commits
 - Github / Bitbucket Integration (Pull-Requests etc.)
 - dark color scheme!
 - fullscreen mode!
 
Lizenz und organisatorisches

 - Community Edition ist kostenlos
 - Pro Edition kostet $6 / Monat
 - man muss sich bei GitKraken noch mal extra anmelden :-(
 - GitKraken Pro hat zusätzlich noch folgende Features:
   - mehrere Profile
   - einen brauchbaren Merge-Editor. Bei der kostenlosen kann man nur zeilenweise mergen - es ist aber möglich externe merge-tools zu konfigurieren
